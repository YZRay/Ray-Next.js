## Dynamic Routes

延續上篇

當頁面需要一些細節頁等等之類的東西，只是資料不同，但是頁面的結構跟元件都一樣時，就需要動態路由來讓頁面可以動態的呈現資料

用法也很簡單，首先得先到 App.js 裡面設定好路由

重點是前面的冒號 : 這個可以讓 react-router 去尋找你設定的值，他告訴 react-router 這個是動態的路由

```js
const router = createBrowserRouter([
  {
    path: "/",
    element: <RootLayout />,
    children: [
      ...,
      {
        path: "/products/:id",
        element: <ProductDetailPage />,
      },
    ],
  },
]);
```

接下來就是到頁面中去取得 id

react-router 中有個方法(useParams)，先把他 import 進來

可以把他 console.log 出來看看，後面的.id 就是你在 App.js 設定的，也就是:後面的值

```js
import { useParams } from "react-router-dom";

const ProductDetailPage = () => {
  const params = useParams();

  console.log(params.id);
  return <div>ProductDetailPage</div>;
};

export default ProductDetailPage;
```
